# **App Name**: EvalFlow

## Core Features:

- Review Period Management: Allows administrators to define review periods with start and end dates, serving as the time boundary for all performance-related activities.
- Goal Plan Configuration: Allows administrators to define goal plans and link them to review periods.
- Document Type Setup: Allows administrators to define different document types, such as Performance Documents, Survey Documents and Feedback Documents.
- Document Section Setup: Allows administrators to define and configure sections for performance documents, including Performance Goals and Overall Summary and sections for Survey & Feedback documents.
- Performance Document Setup: Integrates review periods, goal plans, document types, and document sections to create a live performance document.
- Evaluation Flow Setup: Sets up the performance review lifecycle, assigning tasks to defined roles and controlling the flow using sequencing and parallel modes, with optional time limits on each stage.
- Eligibility Criteria Definition: This section defines rules for excluding employees from receiving the performance document.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6), representing trust, stability, and professionalism.
- Background color: Very light blue (#E3F2FD), providing a clean and calming backdrop.
- Accent color: Light green (#AED581), used for CTAs and positive feedback indicators.
- Headline font: 'Space Grotesk' sans-serif for a modern and clean header and labels.
- Body font: 'Inter' sans-serif providing readability, ensuring clear communication throughout the application
- Single-page configuration following a clear top-down dependency model, ensuring a smooth admin flow.
- Subtle transitions between configuration sections to provide a seamless user experience.
